Welcome to my ECIS project! 

To start, run these files in the specified order:

	python generate_keys.py - this will generate you're keys and will save these files to you're local machine
	
	python encrypt_blob.py - running this will encrypt the .jpg file in your local directory. JPG FILE WILL NEED TO BE SPECIFIED ON LINE 50 IN "encrypt_blob.py" FILE. encrypted file will be stored in your local file.

	python decrypt_blob.py - will decrypt JPG file in your local directory.
